# Задание 2

## Задача 1

### Текст задачи

Найти первую цифру натурального числа.

### Алгоритм решения

1. Попадаем в функцию main

2. Запрашиваем у пользователя натуральное число:

Выводим сообщение: "Введите натуральное число"

Считываем введенное значение в значение n

3. Проверяем корректность ввода:

Если n <= 0, выводим сообщение "Ошибка, число должно быть натуральным" и завершаем программу

Если n > 0, продолжаем выполнение

4. Вызываем рекурсивную функцию firstDigit с аргументом n для нахождения первой цифры:

Передаем значение n в функцию

5. В функции firstDigit (рекурсивная):

Базовый случай: Если число n < 10 (однозначное число), возвращаем само число n — это и есть первая цифра

Рекурсивный случай: Если число n >= 10 (многозначное число), вызываем эту же функцию firstDigit с аргументом n / 10 (отбрасываем последнюю цифру)

Процесс повторяется, пока не дойдем до первой цифры

6. Возвращаемся в функцию main:

Получаем результат result от функции firstDigit

Выводим сообщение: "Первая цифра числа X: Y", где X — исходное число, Y — найденная первая цифра

7. Завершаем программу с кодом возврата 0
